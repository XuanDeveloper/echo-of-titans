# 🎮 Projeto: Echo of Titans 

## 📜 Descrição
Echo of Titans é um jogo **boss rush** inspirado em **Titan Souls**, onde o jogador enfrenta chefões épicos em batalhas estratégicas e desafiadoras. O objetivo é derrotar todos os inimigos utilizando **apenas um ataque**, recuperável após cada lançamento. O jogo busca oferecer um desafio equilibrado, sendo acessível para jogadores casuais e experientes.

## 🚀 Tecnologias Utilizadas
- **Godot 4** (Engine de desenvolvimento)
- **GDScript** (Linguagem de programação)
- **Aseprite/Piskel** (Pixel art para sprites e animações)
- **Bfxr/Freesound** (Efeitos sonoros e trilha sonora)

## 🛠 Como Rodar o Projeto
```bash
# Clone este repositório
git clone https://github.com/XuanDeveloper/echo-of-titans.git

# Entre no diretório do projeto
cd seuprojeto

# Abra com a Godot Engine
```
